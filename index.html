Robot Navigation Simulation I built in Matlab

This code demonstrates a robot navigating through an area with obstacles. The goal for the robot is the green dot in the upper right of the figure. The robot does not know the location of the obstacles until it is next to the obstacle. The robot only knows its current location and the location of its goal.







Robot Arm Simulation in Matlab

This was a group project that required us to simulate a robot loading and unloading nuclear fuel reactors. The main goal of this was to develop the inverse kinematics for the robot. In the code all the paths are paths of the position of the end of the robot arm. The code is constantly calculating the inverse kinematics for how the arm should be positioned to obtain the position of the end of the arm.
